## Installation

###### Requirements

<ol>
  <li>Install Java 8.2</li>
  <li>Install Netbeans</li>
  <li>Install PostgreSQL(pgAdmin)</li>
  <li>Create the database (Arq.: Scrip_Banco_lavaJato.sql)</li>
</ol>


```
Git Clone https://github.com/RuanMiniguite/JavaFX-LavaJato.git
```
